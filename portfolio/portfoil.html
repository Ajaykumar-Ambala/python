Projects :

Analysis of bank marketing : The bank direct marketing campaign for offering products that meet the customer's needs are the challenge problems. The bank direct marketing data analysis is important work that helps the banks predict whether customers will sign a long-term deposit with the banks. The method that can predict such customer's needs can be profitable to the banks for improving their marketing campaign strategies. This project is based on machine learning using python where we use classification machine learning algorithms to predict whether customers will sign a long-term deposit with the banks. Unfortunately, it is very hard to predict the customers' need because the available information is asymmetric. In this presentation, we propose the method to analyses asymmetric information using SMOTE algorithm and Random Forest, Logistic regression. The SMOTE method is used to modify the data and improve the accuracy of the prediction.
